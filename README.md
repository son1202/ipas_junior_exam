# 🧠 IPAS AI 應用規劃師 (初級) 知識庫與題庫

這是一個專為準備 **「經濟部 IPAS 人工智慧應用規劃師 (初級)」** 所建立的系統化開源知識庫。
本專案基於 **Obsidian** 雙向連結筆記軟體架構打造，將龐雜的 AI 考驗範圍拆解為適合碎片化閱讀的學習系統，並結合專屬的模擬測驗引擎。

---

## 🎯 專案核心特色

本資料庫跳脫傳統教科書單調的條列式說明，採用「原子化卡片」與「深水區專文」交織的網狀架構：

1. **雙視角解析法**：每一項 AI 核心觀念皆同時提供「初學者/非技術白話文」與「深度專業底層技術」兩種視角，兼顧跨部門溝通與技術原理掌握。
2. **完美相容 Obsidian**：所有內建連結皆採用 `[[ ]]` 雙向連結格式，可以直接整包匯入至 Obsidian，瞬間建立您的 第二大腦 (Second Brain)。
3. **無痛延伸閱讀**：從廣泛的「概念卡片」出發，遇到深入技術 (如 Transformer 機制或 MICE 插補法) 時，可無縫點擊跳轉至「專題探討」專文。

---

## 📂 目錄結構與內容說明

本專案主要由以下五大核心區塊構成：

### 1. 🗂️ `概念卡片/` (廣度知識基礎)

涵蓋了 IPAS 考試範圍中必須掌握的廣泛概念。每一張卡片包含：

- 核心概念與分類 (如 AGI, LLM, 提示工程)
- 技術架構與應用領域 (金融、製造、醫療)
- 導入流程、企業風險管理與 MLOps 流程
- **附錄：近 70 題的基礎隨堂測驗 (選擇題)**，幫助快速驗收當下章節。

### 2. 🗂️ `專題探討/` (Deep Dive 深度技術領域)

針對容易混淆、或是需要計算與背後數學原理的高階技術點進行拆解。共計 **18 份獨立專文**，重點涵蓋：

- **機器學習模型原理**：包含 Random Forest, SVM, XGBoost 等監督式演算法，與 K-Means, DBSCAN 等非監督群聚演算法。
- **神經網路架構變革**：CNN, LSTM 閘門機制、殘差網路 (ResNet) 到自注意力機制 (Self-Attention / Transformer)。
- **資料工程與預處理實務**：Z-Score 標準化、MICE 多重插補、Isolation Forest 異常檢測與特徵編碼實務。
- **GenAI 落地技術**：RAG 檢索增強生成、LoRA 參數高效微調與 Multi-Agent 工具調用。
- **國際法規規範**：《歐盟 AI 法案 (EU AI Act)》風險金字塔規範。

### 3. 🗂️ `模擬考題/` (實戰驗收)

- 依據進階專題探討所涵蓋的痛點，以 Python 腳本動態生成的 **50 題嚴選進階模擬考卷**。
- 提供便於作答的 `[  ]` 填空框。
- 解答與詳解全面隔離至卷末，徹底模擬真實測驗體驗。

### 4. 🗂️ `觀念釐清/` (錯題本與概念重塑)

- 專屬的考題除錯收納資料夾。當您將寫完的考卷交由 AI 批改後，AI 會針對您答錯的題型進行「盲點突破」，並產出單獨的還原筆記存放置此。
- 每一份錯題筆記皆會重新附上跳轉至《專題探討》的複習連結，打造封閉且高效率的學習迴圈。

### 5. 🗂️ `SKILL_工作流/` (自動化 AI 助教指令)

收錄了讓大語言模型 (LLM) 化身為專業課輔助教的標準化 Prompt (SOP) 文件：

- `生成考題_SKILL.md`：指示 AI 如何根據您的筆記，產出帶有 `[  ]` 填答框且解答完美隔離的精美模擬考卷。
- `考題訂正_SKILL.md`：指示 AI 如何幫您批改考卷、算分，並嚴格遵循「錯題重塑」格式產出專屬的觀念釐清筆記至指定資料夾。
- **使用方法**：若您未來擴充了新筆記，可以直接將對應的 SKILL 檔案作為 Prompt 餵給各式 AI 工具，要求它們「依照這份 SOP 執行代辦事項」。

## 🚀 如何使用 (How to Use)

1. **下載或 Clone 此專案**：將專案從 GitHub 抓取下來 (`git clone https://github.com/son1202/ipas_junior_exam.git`)。
2. **設定 Obsidian 知識庫**：
   - 打開 Obsidian 軟體。
   - 點選 `開啟已有資料夾作為儲存庫 (Open folder as vault)`。
   - 選擇下載的 `ipas_junior_exam` (即本專案的根目錄)。
3. **首頁推薦起點**：
   - 強烈建議優先開啟根目錄下的 `AI應用規劃能力鑑定_MOC.md` (Map of Content)。這是一張總覽地圖，有助於您依照學習藍圖按圖索驥！
