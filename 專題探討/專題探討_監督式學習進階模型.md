# 📊 專題探討：監督式學習進階模型

## 📝 核心概念解析

在「監督式學習 (Supervised Learning)」的領域中，當單純的線性迴歸 (Linear Regression) 或邏輯迴歸 (Logistic Regression) 無法精確捕捉資料中複雜的非線性關係時，我們必須依賴進階的機器學習模型。在「進階延伸研究」中，這三大演算法幾乎統治了所有表格式資料 (Tabular Data) 的預測競賽與企業應用。

### 1. 隨機森林 (Random Forest)
決策樹 (Decision Tree) 是非常直觀的演算法，就像人類玩「猜猜我是誰」一樣，用一連串「是/否」的條件來分類資料。但單一棵樹很容易「死背答案 (過度擬合 Overfitting)」。
- **原理 (Bagging 裝袋法)**：
  - 隨機森林的核心精神是**「三個臭皮匠，勝過一個諸葛亮」**。
  - 它同時種下幾百棵、甚至幾千棵不同的決策樹。為了確保每棵樹長得不一樣，演算法會「**隨機抽取**」一部分的考題（特徵）與學生（資料樣本）給不同的樹去學習。
- **預測方式**：當遇到新的預測任務時，這幾千棵樹會進行「**多數決投票 (Voting)**」。如果有 800 棵樹認為這個病人有糖尿病，200 棵認為沒有，最終結果就是「有」。這種透過群體智慧抵銷單一極端誤差的特性，讓隨機森林極其穩定且抗雜訊力強。

### 2. SVM (支援向量機, Support Vector Machine)
SVM 是一種優雅且數學性極強的分類演算法，特別擅長處理特徵維度極高、且樣本數不多的嚴苛場景。
- **原理**：SVM 的目標是在空間中畫出一條（或一面）**「最佳邊界線 (Hyperplane)」**，把不同類別的資料完美切開。
- **支援向量 (Support Vectors)**：它不管那些離邊界很遠的安全資料，它只專注於那些「離邊界最近、最容易被搞混」的極端點。這些極端點就被稱為「支援向量」。SVM 致力於把這條邊界線畫在距離兩方極端點「**最寬敞的安全車道 (Maximum Margin)**」正中央。
- **核技巧 (Kernel Trick)**：如果資料在 2D 平面上像甜甜圈一樣包在一起，根本畫不出一條直線切開它怎麼辦？SVM 透過神奇的 Kernel Trick，瞬間把資料「打飛」投射到 3D 高維度空間中，然後輕鬆地用一刀 3D 平面切開它們。

### 3. XGBoost (極限梯度提升樹)
如果說隨機森林是各做各的菁英小組，那麼 XGBoost 就是一支**「知錯能改、瘋狂追殺錯誤」的魔鬼特種部隊**。它是 Kaggle 等全球資料科學競賽中最常勝出的霸主。
- **原理 (Boosting 提升法)**：
  1. 種下第一棵樹，先大致預測一個結果。
  2. 檢查這棵樹「**猜錯了多少 (殘差 Residuals)**」。
  3. 種下第二棵樹，但這棵樹的目標不再是預測答案，而是專門去預測「**第一棵樹犯下的錯誤**」，並試圖修正它。
  4. 種下第三棵樹，專門修正第二棵樹留下的錯誤... 依此類推。
- **優勢**：透過這樣幾百層的「梯度提升 (Gradient Boosting)」，模型的誤差會被逼近到物理極限。同時，XGBoost 加入了極其強大的「正則化 (Regularization)」來防止過度擬合，並支援分散式平行運算，運作速度異常狂暴。

## 💡 實務應用場景示範

### 場景一：半導體晶圓瑕疵的超高維度檢測 (SVM)
晶圓廠的生產機台每秒會吐出上萬種感測器參數 (如特定位置的微觀溫度、氣壓震動頻率)。工程師希望找出是哪些極其罕見的參數組合導致了良率下降。
**👉 企業應用**：面對這種「特徵維度破萬，但真正瑕疵晶圓極少」的特殊數據，一般的演算法會直接崩潰或找不到重點。資料科學家導入 **SVM (支援向量機)**。利用其特有的 **Kernel Trick**，SVM 在千萬種可能性的高維度數學空間中，精準地描繪出了一條隔絕正常機台與異常機台的「最大安全邊界」，成功將瑕疵檢出率提升了 15%。

### 場景二：全球保險業的信用違約與定價核保系統 (XGBoost)
保險公司想預測一個來投保車險的 30 歲年輕人，未來一年內出車禍並申請巨額理賠的機率。
**👉 企業應用**：保險業受到金管會高度監管，演算法不能是「黑盒子」。此時，效能與解釋性兼具的 **XGBoost 極限梯度提升樹** 成為最佳解。
一方面，它能處理包含性別、車色、職業、過去違規次數等所有亂七八糟的混合遺失數據，並榨出極限破表的預測準確率；另一方面，XGBoost 內建了強大的 **Feature Importance (特徵重要性)** 輸出功能。它能明確列出一份報表告訴精算師：「『過去違規次數』佔了 45% 的影響力，『職業』佔了 20%」。有了這份清晰的歸屬證據，保險公司才能安穩地向客戶解釋為何他的保費比別人貴了三成。

## 🎯 總結
在預測未知的監督式學習旅途中，這三座大山各有千秋：
追求極致的穩定與自動免疫雜訊，我們選用 **Random Forest (隨機森林)**；
面對高維空間特徵與非線性切分的難題，我們依賴 **SVM (支援向量機)** 的優雅空間映射；
當企業需要在極度殘酷的精確度競爭中奪冠、並榨乾資料每一滴價值時，**XGBoost** 將是我們永不妥協的終極武器。
