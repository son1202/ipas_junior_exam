# 📊 專題探討：高階缺失值填補與異常檢測演算法

## 📝 核心概念解析

在機器學習與資料清洗的實務中，對於**遺失值 (Missing Values)** 與**異常值 (Outliers)** 的處理手法，早已超越了單純的「刪除 (Dropna)」或「補平均數 (Fill Mean)」。進階的處理手法能將資料流失的心血降到最低，並挖掘出隱藏的訊號。

### 1. 高階缺失值填補技術 (Advanced Imputation)
如果資料庫裡某個病人的「血壓」數值空了，單純填補全醫院的平均血壓是非常危險且粗糙的。我們應該根據這個病人的年齡、體重來「合理預測」他的血壓。
- **KNN Imputation (K-近鄰插補法)**：
  - **原理**：找幾個跟他「整體特徵最像」的鄰居。如果這張資料表中，有其他 5 個病人的年齡、體重、飲食習慣都跟他很接近，那就取那 5 個病人的血壓平均值來幫他填補。
- **MICE (多重插補法, Multiple Imputation by Chained Equations)**：
  - **原理**：這是一種更精確的「建模預測」法。把「血壓」當成機器學習方程式裡的 $Y$，其他已知特徵當作 $X$ 建立迴歸模型，算出最適當的血壓值填補進去。這個過程會不斷針對所有遺失的特徵進行「鏈式循環」迭代，直到所有數字穩定為止。

### 2. Isolation Forest (孤立森林) 的異常檢測
異常值檢測 (Anomaly Detection) 不再只是依靠傳統的 Z-Score (看數值離平均值多遠)，因為真實世界的資料通常是高維度且分佈不均勻的。
- **機制**：Isolation Forest 不是利用正向邏輯去「圈出正常人」，而是利用反向邏輯去「**孤立怪咖**」。
- **運作原理**：
  1. 在特徵空間中隨機畫一刀 (Random Split)，把資料切成兩半。
  2. 不斷隨機切畫，直到每一個資料點都被單獨切進一個獨立的空間裡。
  3. **判斷標準**：那些「正常」且簇擁在一起的資料，你要切非常非常多刀才能把它們完全分開；但那些邊緣的「極端異常值 (Outliers)」通常只需要**極少數幾刀**就會被單獨孤立出來。
  4. 最終，模型判定**「被孤立所需要的切割路徑越短者，就是異常值」**。

## 💡 實務應用場景示範

### 場景一：醫療 IoT 儀器的感測器斷訊重建 (KNN Imputation)
加護病房的心跳監測系統可能因為網路波動，每小時會有零星幾秒的資料遺失。
**👉 企業應用**：如果遇到斷訊直接把那幾秒鐘的資料刪掉 (Drop)，會導致後續 LSTM 預測模型無法讀取連續時間序列。這時導入 **KNN Imputation**，從該病患前後十分鐘的歷史波型，或相似病歷病人的波型中尋找「最近鄰」，就能平滑且精確地接續並填補這幾秒鐘的空白，確保訓練出的生病預警模型依然高準度。

### 場景二：跨國電商的高維度防盜刷系統 (Isolation Forest)
跨國信用卡的盜刷特徵不是單一維度能決定的，往往是「深夜 + 買遊戲點數 + IP 在國外」等多維度交錯才會發生。傳統的 Z-Score 演算法面對這種多維立體空間的異常會徹底失效。
**👉 企業應用**：使用 **Isolation Forest (孤立森林)** 將每一筆交易的所有數百個特徵描繪在空間中。因為盜刷行為的模式比起正常消費是非常「稀疏」且與眾不同的，它們在孤立森林隨機畫切砍特徵樹時，會非常輕易地在極淺的樹層 (Short Path Length) 被過濾出來。這使得該演算法成為金融界實時攔截百萬筆交易中的最佳無監督利器。

## 🎯 總結
資料工程師的價值在於不浪費任何一滴辛苦蒐集來的資料。利用 **KNN 與 MICE** 等演算法，我們放棄了粗暴暴力的平均數填補，轉向「為特徵量身定做」的精緻預測；而 **Isolation Forest** 的發明更是打破了傳統統計學依賴常態分佈尋找極端值的弱點，使得高維度、海量空間下的異常行為無所遁形。
